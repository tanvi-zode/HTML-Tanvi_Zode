Hyper text markup language or html is the standard markup language for documents designd to be displayed in a web browser.


Web browsers receive html documents from a web server or from local storage and render the documnts into multimedia web pages.


HTML describes the structure of a web page semantically and originally included cues for its appearance.


HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects such as interactive froms may be emnedded into the rendered page.


Html provide a means to create structured documents by denoting structural semantics for text such as headings, paragraphs,lists,links,and others items.
